## 6. LinkedIn Project Post Draft

I completed a Machine Learning portfolio prototype for logistics driver
matching and recommendation. The system applies vehicle, capacity, and
availability rules before ranking feasible candidates with a scikit-learn
model. I used synthetic data, Python, SQL/SQLite, feature engineering,
shipment-grouped evaluation, classification and ranking metrics, FastAPI,
Streamlit, pytest, and GitHub Actions.

The project is a learning and demonstration prototype rather than a production
dispatch system. It uses no real company, customer, driver, or shipment data,
and the reported metrics describe only the synthetic held-out dataset.
